# 安全公開ポリシー

このリポジトリは公開用OSSとして使うため、秘密情報や実店舗の内部情報を含めないことを重視します。

## 公開してよい情報

- テンプレートの項目名
- 架空の商品データ
- 一般的な作業手順
- 公開しても問題ない説明文
- 利用条件を確認済みの画像に関するメモ
- 具体的な店舗名や顧客情報を含まないノウハウ

## 公開してはいけない情報

- APIキー、トークン、秘密鍵
- `.env` の内容
- 顧客情報
- 注文情報
- メールアドレス、電話番号、住所などの個人情報
- 原価、仕入れ価格、非公開仕入れ先情報
- BASE管理画面の非公開情報
- BASE本番環境の設定情報
- Cloudinaryなど外部サービスの認証情報や本番設定
- 実在の顧客、注文、社内メモ

## サンプルデータのルール

- サンプル商品はすべて架空にする
- 実在メーカー名を使わない
- 実在JANを使わない
- 実在商品画像URLを使わない
- 画像URLが必要な場合は `example.invalid` などの無効な例示ドメインを使う
- 店舗の内部情報や売上情報を入れない

## 画像のルール

- 自分で撮影した画像、利用許可を得た画像、または利用可能なライセンスの画像だけを使う
- 画像の出典を確認する
- 権利不明画像は使わない
- 他店の商品ページやSNSから無断取得した画像は使わない
- 個人情報、配送伝票、住所、注文番号が写った画像は使わない

## 公開前チェックリスト

- [ ] `.env` が含まれていない
- [ ] APIキーやトークンが含まれていない
- [ ] 顧客情報、注文情報、住所、電話番号、メールアドレスが含まれていない
- [ ] 原価や非公開仕入れ情報が含まれていない
- [ ] BASE管理画面の非公開情報が含まれていない
- [ ] Cloudinaryなど外部サービスの本番情報が含まれていない
- [ ] サンプル商品がすべて架空である
- [ ] 実在JANや実在商品画像URLが含まれていない
- [ ] Markdownの表示が崩れていない
- [ ] CSVの列数とヘッダーが揃っている
- [ ] `git diff --check` に問題がない
