# BASE EC Product Template JA

This repository provides practical templates for small retail stores using BASE.

It helps organize product listings, SEO keywords, product descriptions, product images, image source checks, JAN codes, color variations, and stock.

It is designed especially for small shops such as fishing tackle stores where product colors, JAN codes, and image management can become complicated.

This project does not require API keys, customer data, order data, or private store data. It is intentionally limited to public-safe templates, fictional sample data, and operational checklists.

It can be used together with coding assistants such as Codex to prepare safer product listing workflows, review CSV files, draft product descriptions, and improve documentation before product registration.

This is a small but practical open-source project based on real operational needs from a small retail store.

## What Is Included

- Product registration checklist
- Product description template
- SEO keyword template
- SEO description template
- Image source checklist
- JAN, color, and stock CSV template
- Fictional sample CSV files
- Safe publication policy
- Application notes for OpenAI Codex for Open Source
- CSV validation policy for public-safe examples

## Who This Is For

- Small retail stores using BASE
- Fishing tackle shops and similar stores with many color variations
- Non-engineer operators who want simple product listing workflows
- Maintainers who want public-safe templates that avoid customer data and secrets

## Safety Principles

- Do not include API keys, tokens, `.env` files, customer data, order data, cost data, or private supplier information.
- Use only fictional sample products in public repositories.
- Use only images with confirmed permission, public license, or your own original rights.
- Do not publish BASE admin screen information or production store data.
