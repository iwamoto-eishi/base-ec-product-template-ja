# BASE EC 商品登録テンプレート集

小規模店舗・釣具店向けの、BASE商品登録テンプレート集です。

商品名、メーカー、JAN、カラー、在庫、価格、画像、SEOキーワード、SEO説明文を、商品登録前に整理しやすくするためのMarkdownとCSVテンプレートをまとめています。

非エンジニアでも使いやすいように、専門用語を少なめにし、実際の作業前チェックに使える形を目指しています。

このリポジトリでは、APIキー、顧客情報、注文情報、原価情報、非公開仕入れ情報などは扱いません。公開しても安全なテンプレートと、すべて架空のサンプルデータだけを置きます。

Codexなどのコード支援ツールと組み合わせることで、商品登録前の確認、CSVの見直し、SEO文章の下書き、チェックリスト改善に使えます。

まだ小規模なOSSですが、実店舗運営の現場課題から作った実用的なテンプレート集です。

## 入っているもの

- 商品登録前チェックリスト
- 商品説明文テンプレート
- SEOキーワードテンプレート
- SEO説明文テンプレート
- 画像出典チェックリスト
- JAN・カラー・在庫整理用CSV
- 架空商品のサンプルCSV
- 安全公開ポリシー
- OpenAI Codex for Open Source 申請用メモ

## 使い方

1. `templates/` のテンプレートをコピーします。
2. 商品名、カラー、JAN、在庫、画像出典などを入力します。
3. 公開前に `docs/safe-publication-policy.md` のチェックリストを確認します。
4. 必要に応じてCodexなどの支援ツールで、文章やCSVの整合性を確認します。

## 注意

- 実在の顧客情報、注文情報、原価、APIキー、トークンは入れないでください。
- サンプルデータは必ず架空にしてください。
- 権利が不明な画像URLは入れないでください。
- BASE管理画面の非公開情報は公開しないでください。
