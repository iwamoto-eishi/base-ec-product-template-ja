# 画像出典チェックリスト

商品画像を使う前に、出典と利用可否を確認するためのチェックリストです。

- 確認日:

## 利用可否

- [ ] 自社または自分で撮影した画像である
- [ ] メーカーや提供元から利用許可を得ている
- [ ] 利用条件を確認した
- [ ] クレジット表記が必要か確認した
- [ ] 加工やトリミングの可否を確認した

## 使ってはいけない画像

- [ ] 権利者が不明な画像
- [ ] SNSや検索結果から無断保存した画像
- [ ] 利用条件が確認できない画像
- [ ] 他店の商品ページから取得した画像
- [ ] 個人情報や住所、注文情報が写っている画像
