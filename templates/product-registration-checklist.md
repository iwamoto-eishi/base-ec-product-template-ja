# 商品登録前チェックリスト

BASEに商品を登録する前に確認するためのチェックリストです。

## 基本情報

- [ ] 商品名はわかりやすい
- [ ] ブランド名またはメーカー名を確認した
- [ ] カテゴリを確認した
- [ ] 販売価格を確認した
- [ ] 定価または参考価格の扱いを確認した
- [ ] 在庫数を確認した
- [ ] カラー名・サイズ名・規格名を確認した

## JAN・カラー・在庫

- [ ] JANコードを確認した
- [ ] JANがない場合は「なし」または空欄ルールを決めた
- [ ] カラー違いの商品を混同していない
- [ ] 同じ商品名でカラー違いがある場合、色名が明確
- [ ] 在庫数が販売可能数と一致している

## 商品説明

- [ ] 商品の特徴を短く説明している
- [ ] 対象ユーザーや使用シーンがわかる
- [ ] 誇大表現を避けている
- [ ] 注意点や仕様違いがある場合は明記した
- [ ] 架空ではない事実だけを書いている

## 画像

- [ ] 画像ファイル名を確認した
- [ ] 画像の出典を確認した
- [ ] 利用許可または利用条件を確認した
- [ ] 権利不明画像を使っていない
- [ ] 実店舗の内部情報や個人情報が写っていない

## SEO

- [ ] SEOキーワードを3から8個程度に整理した
- [ ] 商品名と関係のないキーワードを入れていない
- [ ] SEO説明文は自然な日本語になっている
- [ ] 同じ文章の使い回しが多すぎない

## 公開前安全確認

- [ ] APIキー、トークン、`.env` を含めていない
- [ ] 顧客情報、注文情報、住所、電話番号、メールアドレスを含めていない
- [ ] 原価や非公開仕入れ情報を含めていない
- [ ] BASE管理画面の非公開情報を含めていない
- [ ] サンプルの場合は架空データであることを確認した
