# SEO説明文テンプレート

検索結果や商品一覧で自然に読める説明文を作るためのテンプレートです。

## 入力項目

- 商品名:
- ブランド:
- カテゴリ:
- 主な特徴:
- おすすめの使用シーン:
- カラーまたは規格:

## 説明文テンプレート

```text
{商品名}は、{主な特徴}を備えた{カテゴリ}です。{おすすめの使用シーン}に使いやすく、{カラーまたは規格}を選べます。
```

## 完成例

```text
架空ミノー 90Fは、扱いやすいサイズ感を備えたルアーです。浅場のチェックや短時間の釣行に使いやすく、複数のカラーを選べます。
```

## 確認項目

- [ ] 120文字前後で自然に読める
- [ ] 実際の商品特徴と一致している
- [ ] 誇大表現を避けている
- [ ] 関係のないキーワードを詰め込んでいない
